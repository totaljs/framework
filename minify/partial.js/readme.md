## Install partial.js step by step

![partial.js on NPM](https://nodei.co/npm/partial.js.png?downloads=true)

```
$ sudo npm install -g partial.js
```
```
$ mkdir mywebsite
$ cd mywebsite
$ partial.js
```
```
$ node debugging

# OR

$ node index 8080
```
```
$ open http://127.0.0.1:8000
```
